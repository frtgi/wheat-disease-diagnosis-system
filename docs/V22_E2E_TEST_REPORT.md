## 三、Flaky 测试分析

### 3.1 登出后 Token 被清除 (auth.spec.ts)

**现象**: 首次运行超时，重试后通过。

**根因**: 登出测试清除 localStorage 后，后续 `login()` 函数的 `waitForFunction` 在 SPA 路由导航时偶尔不触发。这是 Playwright 在 SPA 应用中的已知行为——`waitForFunction` 依赖 `requestAnimationFrame`，在页面非活跃状态下可能不执行。

**缓解措施**: 配置 `retries: 2`，重试后稳定通过。

### 3.2 批量诊断入口可访问 (diagnosis.spec.ts)

**现象**: 同上，首次运行超时，重试后通过。

**根因**: 与 3.1 相同，`loggedInPage` fixture 的 `login()` 函数在连续测试中偶尔超时。

**缓解措施**: 同上。
