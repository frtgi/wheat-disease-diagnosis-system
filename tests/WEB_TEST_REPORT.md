## 🔧 技术细节

### 懒加载策略
所有引擎采用懒加载策略，在后台线程中并行加载：
- 视觉引擎优先加载（用于快速响应图像诊断）
- 知识图谱、融合、认知引擎并行加载
- 加载进度实时显示在 Web 界面

### 模型加载优化
- **YOLOv8**: 使用 Ultralytics 框架，支持 GPU 加速
- **Qwen3-VL**: 使用 4bit 量化，显存优化至~3GB
- **Neo4j**: 持久化连接，避免重复初始化

### 缓存策略
- **推理缓存**: LRU 策略，500 条上限，30 分钟过期
- **MD5 哈希**: 基于图像内容去重
- **命中率监控**: 实时统计缓存命中率
